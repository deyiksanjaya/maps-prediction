Cara Kerja:

Aplikasi ini memiliki dua mode: Pilih Set dan Mode Prediksi.

  1. Saat pertama dibuka, Anda akan melihat set-set prediksi yang tersedia (misal: "1-5", "6-10").
  2. Tekan salah satu set untuk masuk ke Mode Prediksi dan melihat item di dalamnya.
  3. Tekan item yang diinginkan untuk membuka Google Maps/Search/Shortcut.
  4. Untuk navigasi, geser ke kiri untuk melihat set berikutnya. Geser ke kanan untuk kembali ke Pilihan Set (Undo).
Integrasi Lanjutan: Shortcut iOS

Anda bisa menghubungkan aplikasi ini dengan aplikasi Pintasan (Shortcuts) di perangkat iOS untuk aksi yang sepenuhnya kustom. Cara Mengaturnya:

  1. Buat Pintasan di iOS: Buka aplikasi Pintasan, buat pintasan baru.
  2. Aktifkan Input: Di dalam editor pintasan, ketuk `i` atau nama pintasan, lalu aktifkan "Tampilkan di Lembar Berbagi".
  3. Gunakan Input: Gunakan variabel "Input Pintasan" di dalam aksi Anda.
  4. Atur di Aplikasi: Buka Pengaturan (⚙️), pilih "Shortcut iOS", dan masukkan nama Pintasan Anda dengan sama persis.
Jadikan Aplikasi:

Aplikasi ini bisa berfungsi seperti aplikasi native. Tekan tombol 'Bagikan' lalu 'Tambahkan ke Layar Utama' (Safari di iOS), atau tekan titik tiga (⋮) lalu 'Tambahkan ke Layar Utama' (Chrome di Android).
