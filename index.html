Petunjuk Penggunaan

Mode Grid

  1. Layar utama menampilkan halaman set (misal: "1-5"). Geser ke kiri/kanan untuk berpindah halaman set.
  2. Tekan sebuah set untuk melihat isinya.
  3. Saat melihat item, geser ke kiri untuk ke set berikutnya, atau geser ke kanan untuk kembali (Undo).

Mode Pengenalan Suara

Mode pengenalan suara menggunakan struktur kalimat khusus untuk memastikan perintah Anda akurat dan untuk menghindari pemicu yang tidak disengaja dari percakapan sekitar.

  1. Frasa Pemicu: Aplikasi menunggu Anda mengucapkan kalimat lengkap yang terdiri dari [Kata Pembuka] + [ITEM PREDIKSI] + [Kata Penutup].
  2. Cara Menggunakan:
    • Tekan tombol mikrofon. Aplikasi akan meminta izin akses mikrofon saat pertama kali digunakan.
    • Ucapkan kalimat perintah Anda secara lengkap. Contoh default:
      "Yang kamu pilih adalah Monumen Nasional betul?"
    • Aplikasi akan otomatis mendeteksi item ("Monumen Nasional"), berhenti merekam, dan membuka tujuan Anda.
  3. Kustomisasi: Anda dapat mengubah "Kata Pembuka" dan "Kata Penutup" sesuai keinginan Anda di menu Pengaturan (⚙️).

Pengaturan

Di Pengaturan (⚙️), Anda bisa memilih antara mode Grid atau Suara, mengubah frasa suara, mengatur tujuan redirect (Maps, Search, Shortcut), dan mengedit daftar prediksi Anda.
