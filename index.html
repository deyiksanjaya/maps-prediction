Prediksikan ke:

Daftar Prediksi (Bisa Dihapus)

Petunjuk Penggunaan

Cara Kerja:
  1. Saat pertama dibuka, Anda akan melihat set-set prediksi yang tersedia (misal: "1-5").
  2. Tekan salah satu set untuk masuk ke Mode Prediksi.
  3. Tekan item yang diinginkan untuk membuka tujuan.
  4. Geser ke kiri untuk melihat set berikutnya. Geser ke kanan untuk kembali ke Pilihan Set (Undo).
Mengatur Prediksi:
  1. Buka Pengaturan (⚙️).
  2. Ketik semua item prediksi Anda di kotak input besar, pisahkan dengan koma.
  3. Tekan tombol "Proses & Tampilkan Daftar".
  4. Daftar item akan muncul. Anda bisa menghapus item yang tidak diinginkan dengan menekan ikon tong sampah.
  5. Setelah selesai, tekan "Simpan Perubahan".
Jadikan Aplikasi:

Aplikasi ini bisa berfungsi seperti aplikasi native. Tekan tombol 'Bagikan' lalu 'Tambahkan ke Layar Utama' (Safari di iOS), atau tekan titik tiga (⋮) lalu 'Tambahkan ke Layar Utama' (Chrome di Android).
