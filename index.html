Petunjuk Penggunaan

Navigasi Utama

  1. Pilih Halaman Set: Layar utama menampilkan halaman-halaman yang berisi set prediksi (misal: "1-5", "6-10").
    Geser ke kiri atau kanan di layar utama untuk berpindah antar halaman jika Anda memiliki lebih dari 5 set.
  2. Masuk ke Set: Tekan salah satu set (misal: "6-10") untuk melihat item di dalamnya.
  3. Navigasi di Dalam Set: Saat melihat item, geser ke kiri untuk pindah ke set berikutnya (misal: dari set "6-10" ke "11-15"). Geser ke kanan untuk kembali ke layar pemilihan set (Undo).

Pengaturan Aplikasi

  1. Metode Input: Di Pengaturan (⚙️), Anda bisa memilih "Metode Input Utama". Saat ini, hanya "Tampilan Grid" yang tersedia. Fitur "Pengenalan Suara" akan segera hadir, memungkinkan Anda memilih prediksi dengan suara.
  2. Edit Prediksi: Untuk menambah atau mengubah daftar, masukkan semua item Anda di kotak teks besar, pisahkan dengan koma. Lalu tekan "Simpan Perubahan".

Jadikan Aplikasi

Aplikasi ini bisa di-install di layar utama Anda. Tekan tombol 'Bagikan' lalu 'Tambahkan ke Layar Utama' (Safari di iOS), atau tekan titik tiga (⋮) lalu 'Tambahkan ke Layar Utama' (Chrome di Android).
